# Skoolis Connect App

Ce dossier est dédié au développement de l'application mobile et/ou PWA (Progressive Web App) pour **Skoolis**.

## Objectif

L'objectif de ce projet (`skoolis_connect_app`) est de regrouper tout le code et les ressources nécessaires pour créer une application native (iOS/Android) ou une PWA (Progressive Web App), indépendamment de l'application web principale et de la landing page.

## Technologies possibles

Ce dossier pourra accueillir l'initialisation d'un framework mobile tel que :
- **React Native** (avec ou sans Expo)
- **Flutter**
- **Ionic** / **Capacitor**
- **PWA** (Progressive Web App) classique

## Prochaines étapes

1. Choisir la stack technologique (framework, langages).
2. Initialiser le projet à l'intérieur de ce répertoire.
3. Connecter l'application aux API backend existantes de l'écosystème Skoolis.
